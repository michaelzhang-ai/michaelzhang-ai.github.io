# michaelzhang-ai.github.io

Personal portfolio website for Michael Zhang — ML/Systems Engineer specializing in LLM serving infrastructure, GPU optimization, and computer vision.

## Live Site

[michaelzhang-ai.github.io](https://michaelzhang-ai.github.io)

## Structure

```
index.html              Main page (about, experience, skills, publications, patents, service)
projects/
  sglang.html           SGLang CI Infrastructure on AMD GPUs
  text2video.html       Text2Video (ICASSP 2022)
  excavator.html        Excavator Safety Monitoring (ISARC 2021)
  speech2video.html     Speech2Video Synthesis (ACCV 2020)
  dvi.html              Depth Guided Video Inpainting (ECCV 2020)
  trafficpredict.html   TrafficPredict (AAAI 2019)
```

## Features

- Light/dark mode via `prefers-color-scheme`
- Responsive navigation with Projects dropdown
- Mobile hamburger menu with touch-toggle support
- Scroll-reveal animations
- No build step — static HTML/CSS/JS hosted on GitHub Pages
